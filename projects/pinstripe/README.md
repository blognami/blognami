
# Pinstripe JS

Pinstripe is a fullstack JavaScript web framework, that acts as a foundation for Blognami to be built upon.

In order to give Blognami the best possible developer experience Pinstripe has been built from the ground up to provide this.

## Notable Features

### Convention over Configuration

By favouring convention over configuration, Pinstripe encorages DRY code, and makes developers happy and productive by eliminating a lot of arbitrary decision making, and the need to write repetitive boilerplate code.

### Incremental page updates

Pinstripe makes the UI of your web application run much faster, by only updating the parts of the DOM that change when loading html content from your server. Thus removing the performance cost of a full page reload.
