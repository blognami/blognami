# @blognami/pageables

Implements pageables functionality in the main app. This package provides the core pageable model and related functionality for creating URL-routable entities with automatic slug generation.

## Features

- Pageable model with automatic slug generation
- URL routing for pageable entities
- Validation for unique slugs
- Support for pageable views rendering

## Usage

This package is automatically loaded as part of the Blognami application. Pages and Posts extend the pageable functionality to provide URL routing capabilities.
