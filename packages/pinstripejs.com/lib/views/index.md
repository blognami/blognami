# Welcome to Pinstripe JS

Pinstripe JS is a modern, lightweight JavaScript web framework designed for building fast, scalable web applications with minimal configuration.

## Quick Start

Get started with Pinstripe JS in just a few steps:

```bash
npm install -g pinstripe
pinstripe create my-app
cd my-app
npm start
```

### Features

Pinstripe JS comes with everything you need to build modern web applications:

- File-based routing
- Server-side rendering
- Built-in styling system
- Database integration
- Multi-tenant support
